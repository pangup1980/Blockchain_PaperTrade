# Blockchain Paper Trade

A comprehensive paper trading platform for the Indian stock market with blockchain-based trade recording.

## Features

- Real-time market data simulation
- Virtual trading with Indian stocks
- Portfolio management with P&L tracking
- Watchlist
- Order types: Market and Limit
- Trade history with blockchain hashes
- Responsive UI

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Recharts for charts
- Lucide React for icons

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Usage

- Start with ₹100,000 virtual balance
- Select stocks and place buy/sell orders
- Monitor your portfolio and P&L
- View trade history recorded on blockchain

## Blockchain Integration

Trades are recorded with unique hashes simulating blockchain storage for immutability.
